## Introduction
本系统为初学MFC时所开发的，系统每段代码都有详细注释，包含当时自己的一些理解，适合初学者练手用。
系统采用VC6.0平台开发，存储采用SQLite嵌入式数据库，GUI界面设计采用MFC，系统初始密码“123456798”。

## Usage

1. Star 本项目
2. Clone 到本地
3. 用VS6.0打开工程
4. 编译执行
5. 部署上线
. Enjoy it!
